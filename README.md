## 시작하기
아래 명령어를 실행합니다.
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## 접속하기
[http://localhost:3000](http://localhost:3000) 을 브라우저에서 열어서 결과를 확인합니다.
